uint16_t Now ( )

Returns the number of milliseconds since OS_Init(). Note that this number wraps around after it overflows as an unsigned integer. The arithmetic of 2's complement will take care of this wrap-around behaviour if you use this number correctly. Let T = Now() and we want to know when Now() reaches T+1000. Now() is always increasing. Even if Now() wraps around, (Now() - T) always >= 0. As long as the duration of interest is less than the wrap-around time, then (Now() - T >= 1000) would mean we have reached T+1000. However, we cannot compare Now() against T directly due to this wrap-around behaviour. Now() will wrap around every 65536 milliseconds. Therefore, for measurement purposes, it should be used for durations less than 65 seconds.

return time since operation began in millis.

int16_t  v 
)
Parameters
ea Service descriptor

The calling task publishes a new value "v" to service "s". All waiting tasks on service "s" will be resumed and receive a copy of this value "v". Values generated by services without subscribers will be lost.

The calling task waits for the next published value associated with service "s". More than one task may wait for a service. When a new value "v" is published to "s", all waiting tasks resume and obtain the same value.
